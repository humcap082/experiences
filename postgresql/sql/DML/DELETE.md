# DELETE

行を削除する

```sql
DELETE FROM table_name [AS]
    [WHERE]
    [RETURNING]
```


## パラメータ

<details><summary>table_name</summary>

行を削除するテーブルの名前

</details>

## 句

<details><summary>AS</summary>

対象のテーブルや集合に名前をつける。

```sql
AS alias
```

</details>

# cast()

キャストする関数。

```sql
cast(<expression> AS <type>)
```

## 引数

<details><summary>expression</summary>

キャストされる値を入れる。

</details>


<details><summary>type</summary>

`expression`をキャストする型

</details>

## 備考

<details><summary>ショートカット</summary>

変数のうしろにに`::`をつけるとキャストできる。

```sql
<expression>::<type>
```

</details>

# DROP TABLE

テーブルを削除する。

```sql
DROP TABLE [IF EXISTS] <name>[, ...] [CASCADE | RESTRICT]
```

## パラメータ

<details><summary>name</summary>

削除するテーブル名

</details>

## 句

<details><summary>IF EXISTS</summary>

```sql
IF EXISTS
```

</details>

<details><summary>CASCADE</summary>

```sql
CASCADE
```

</details>

<details><summary>RESTRICT</summary>

```sql
RESTRICT
```

</details>
